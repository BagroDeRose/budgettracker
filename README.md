# 💰 BudgetTracker

> Клиент-серверное веб-приложение для учета и анализа личных финансов.  
> Разработано в рамках курсового проекта по дисциплине «Интернет-программирование».

---

## 🛠️ Технологический стек

| Компонент  | Технология                                      |
|------------|-------------------------------------------------|
| **Frontend** | HTML5, CSS3 (Flexbox/Grid, CSS Variables), Vanilla JS (ES6+, Fetch API) |
| **Backend**  | PHP 8.0+ (PDO, Sessions, `password_hash`)     |
| **Database** | MySQL / MariaDB                                 |
| **Server**   | Apache (XAMPP / OpenServer)                     |
| **Libs**     | Chart.js (визуализация данных)                  |

## 🚀 Установка и запуск

### ✅ Требования
- Установленный пакет **[XAMPP](https://www.apachefriends.org/ru/index.html)** или **[OpenServer](https://ospanel.io/)** (PHP 8.0+, MySQL 8.0+)
- Современный браузер (Chrome, Firefox, Yandex)
- Текстовый редактор (VS Code, Sublime Text, Notepad++)

### 📥 Пошаговая инструкция

1. **Скачайте проект**
   - Нажмите кнопку **Code** → **Download ZIP** на GitHub.
   - Либо клонируйте репозиторий:
   ```bash
   git clone https://github.com/BagroDeRose/budgettracker.git
   ```
2. **Разместите файлы на сервере**
   - Скопируйте папку проекта в директорию веб-сервера:
     - Для **XAMPP**: `C:\xampp\htdocs\budgettracker`
     - Для **OpenServer**: `domains\budgettracker`

3. **Настройте базу данных**
   - Запустите модули **Apache** и **MySQL** в панели управления.
   - Откройте **phpMyAdmin**: `http://localhost/phpmyadmin`
   - Создайте новую базу данных с именем `budgettracker` (кодировка `utf8mb4_general_ci`).
   - Перейдите во вкладку **Импорт** и загрузите файл `database/budgettracker.sql`.

4. **Проверьте конфигурацию**
   - Откройте файл `includes/config.php` и убедитесь в корректности параметров:
     ```php
     define('DB_HOST', 'localhost');
     define('DB_NAME', 'budgettracker');
     define('DB_USER', 'root');
     define('DB_PASS', ''); // Пароль по умолчанию пустой
     ```

5. **Запуск приложения**
   - Откройте браузер и перейдите по адресу:
     🔗 `http://localhost/budgettracker`

## ✨ Реализованный функционал

В рамках курсового проекта были реализованы следующие возможности:

### 🔐 Безопасность и Авторизация
- [x] **Регистрация и Вход**: Безопасное создание учетных записей с валидацией email.
- [x] **Защита сессий**: Использование `session_regenerate_id` для предотвращения фиксации сессии.
- [x] **Хеширование паролей**: Алгоритм `bcrypt` через `password_hash`/`verify`.
- [x] **Защита от SQL-инъекций**: Все запросы реализованы через подготовленные выражения **PDO**.
- [x] **XSS-фильтрация**: Экранирование пользовательских данных через `htmlspecialchars`.

### 📊 Аналитика и Статистика
- [x] **Дашборд**: Отображение текущего баланса, сумм доходов и расходов в реальном времени.
- [x] **Визуализация**: Интерактивная круговая диаграмма структуры расходов (библиотека **Chart.js**).

### 💼 Управление финансами
- [x] **CRUD Операции**: Создание, чтение, обновление и удаление транзакций.
- [x] **Категории**: Гибкая система категорий для доходов и расходов с возможностью добавления своих.
- [x] **Фильтрация**: Поиск по описанию, выборка по периодам, типу и категории.

### 🎨 Интерфейс и UX
- [x] **Темная и Светлая темы**: Переключение тем оформления с сохранением в сессии.
- [x] **Денежная тема**: Специальная тема с фоновым паттерном и акцентными цветами.
- [x] **Адаптивность**: Корректное отображение на мобильных устройствах (Flexbox/Grid).
- [x] **Микроанимации**: Плавные переходы и акцентные эффекты при взаимодействии.

### 📥 Экспорт и Печать
- [x] **CSV Экспорт**: Генерация отчета в формате CSV для открытия в Excel (с поддержкой UTF-8 BOM).
- [x] **Версия для печати**: Оптимизированный вид (`@media print`) для вывода отчета на бумагу.
