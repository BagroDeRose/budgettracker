# 💰 BudgetTracker

> Клиент-серверное веб-приложение для учета и анализа личных финансов.  
> Разработано в рамках курсового проекта по дисциплине **«Интернет-программирование»**.

---

## 📖 Описание проекта

**BudgetTracker** — это современная система для контроля доходов и расходов. Приложение позволяет пользователям вести журнал операций, категоризировать траты, анализировать финансовое состояние с помощью интерактивных графиков и экспортировать отчеты.

Проект разработан с упором на **безопасность** (защита от SQL-инъекций, XSS, хеширование паролей), **адаптивность** (Mobile-First) и **чистую архитектуру** кода без использования тяжелых фреймворков.

---

## ✨ Реализованный функционал

| Функция | Описание |
| :--- | :--- |
| 🔐 **Безопасность** | Авторизация/Регистрация, `bcrypt` хеширование, защита сессий, CSRF-защита |
| 📊 **Аналитика** | Расчет баланса, диаграмма структуры расходов (Chart.js) |

| 🔍 **Фильтрация** | Поиск операций по дате, типу, категории и описанию |
| ✏️ **CRUD** | Полное управление операциями и категориями (Create, Read, Update, Delete) |
| 🖨️ **Печать** | Оптимизированная версия для печати (`Ctrl+P`) |

## 🛠️ Технологический стек

Проект реализован на классическом стеке **LAMP** (без тяжелых фреймворков, демонстрируя глубокое понимание основ):

*   **Frontend:** HTML5, CSS3 (Flexbox/Grid, CSS Variables), Vanilla JS (ES6+, Fetch API).
*   **Backend:** PHP 8.0+ (PDO, Sessions, JSON API).
*   **Database:** MySQL / MariaDB.
*   **Server:** Apache (XAMPP / OpenServer).
*   **Libraries:** Chart.js (визуализация данных).

## 🚀 Установка и запуск

### ✅ Требования

- Установленный пакет **XAMPP** или **OpenServer**.
- **PHP 8.0+** и **MySQL 8.0+**.
- Современный браузер (Chrome, Firefox).

### 📥 Пошаговая инструкция

1. **Скачайте проект**  
   ```bash
   git clone https://github.com/your-username/budgettracker.git

2. **Разместите файлы на сервере**  
   Переместите папку `budgettracker` в директорию веб-сервера:  
   - Для XAMPP: `C:\xampp\htdocs\budgettracker`  
   - Для OpenServer: `domains\budgettracker`

3. **Настройте базу данных**  
   - Запустите модули **Apache** и **MySQL**.  
   - Откройте phpMyAdmin: [http://localhost/phpmyadmin](http://localhost/phpmyadmin)  
   - Создайте базу данных с именем `budgettracker` (кодировка `utf8mb4_general_ci`).  
   - Перейдите во вкладку **Импорт** и загрузите файл `database/budgettracker.sql`.

4. **Проверьте конфигурацию**  
   Убедитесь, что в файле `includes/config.php` параметры подключения совпадают с вашей средой (обычно стандартные для XAMPP):  
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'budgettracker');
   define('DB_USER', 'root');
   define('DB_PASS', '');

5. **Запуск**
Откройте в браузере:
🔗 http://localhost/budgettracker
